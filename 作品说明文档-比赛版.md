## 二、作品简介

### 2.1 背景与问题

高校学生社团是"第二课堂"育人的重要载体，但目前社团管理普遍面临以下痛点：

**（1）活动效果难以科学评估**
- 传统评价依赖主观感受，缺乏量化指标
- 评价维度单一，无法全面反映活动质量
- 评价结果利用率低，难以指导后续改进

**（2）资源配置效率低下**
- 场地、资金分配依赖经验判断
- 供需信息不对称，资源利用率不高
- 缺乏数据支撑，决策科学性不足

**（3）反馈信息处理困难**
- 学生评价数据量大、格式多样
- 人工分析耗时耗力，容易遗漏关键信息
- 情感倾向难以准确把握

**（4）管理流程繁琐低效**
- 活动申报、审批、总结等环节信息化程度低
- 多端信息孤岛，协同效率差
- 历史数据难以沉淀复用

### 2.2 解决方案

本系统构建了一套基于AI算法的社团活动全生命周期管理平台，核心创新点：

- **五维评估体系**：基于AHP层次分析法，构建参与度、教育性、创新性、影响力、可持续性五个维度的量化评估模型
- **AI智能分析**：集成NLP情感分析、K-Means聚类、LSTM预测等多种算法，实现数据深度挖掘
- **资源优化配置**：智能调度场地、资金等资源，提升使用效率
- **三端协同平台**：学生端（UniApp）、社团端（Vue3）、管理端（Vue3）数据互通

### 3.3 核心算法

#### 3.3.1 AHP层次分析法

**评估维度与权重**：

| 维度 | 权重 | 说明 |
|------|------|------|
| 参与度 | 0.32 | 报名人数/容量比、实际到场率 |
| 影响力 | 0.22 | 传播范围、后续关注度 |
| 教育性 | 0.18 | 知识传递度、技能提升度 |
| 创新性 | 0.15 | 活动形式新颖度、内容创意度 |
| 可持续性 | 0.13 | 可复制性、品牌化潜力 |

**一致性检验**：
- 构建5阶判断矩阵
- 计算最大特征值 λmax
- 一致性指标 CI = (λmax - n) / (n - 1)
- 一致性比率 CR = CI / RI < 0.1（通过检验）

#### 3.3.2 NLP情感分析

基于SnowNLP实现中文文本情感分析：

```python
def analyze_sentiment(text: str) -> SentimentResult:
    s = SnowNLP(text)
    score = s.sentiments  # 0-1之间的情感分值
    
    if score > 0.6:
        level = "POSITIVE"  # 正面
    elif score < 0.4:
        level = "NEGATIVE"  # 负面
    else:
        level = "NEUTRAL"   # 中性
    
    # 提取关键词
    keywords = s.keywords(5)
    
    return SentimentResult(score, level, keywords)
```

#### 3.3.3 K-Means聚类分析

对历史活动进行聚类，识别活动特征模式：

```python
from sklearn.cluster import KMeans
from sklearn.preprocessing import StandardScaler

# 特征向量：[规模, 预算, 时长, 类型编码]
features = [[scale, budget, duration, type_code], ...]

# 标准化
scaler = StandardScaler()
X_scaled = scaler.fit_transform(features)

# K-Means聚类（K=4）
kmeans = KMeans(n_clusters=4, random_state=42)
labels = kmeans.fit_predict(X_scaled)
```

聚类结果可用于：
- 发现相似活动群体
- 资源分配参考
- 异常活动检测

#### 3.3.4 LSTM参与度预测

基于历史数据预测活动参与人数：

```python
import torch.nn as nn

class ParticipationLSTM(nn.Module):
    def __init__(self, input_size, hidden_size, num_layers):
        super().__init__()
        self.lstm = nn.LSTM(input_size, hidden_size, 
                           num_layers, batch_first=True)
        self.fc = nn.Linear(hidden_size, 1)
    
    def forward(self, x):
        out, _ = self.lstm(x)
        out = self.fc(out[:, -1, :])
        return out
```

输入特征：历史参与人数、活动类型、时间因素、宣传力度等

- 三端数据一致性保障机制
- 离线缓存与增量同步

### 5.2 应用创新

**（1）量化评估模型创新**
- 首次将AHP层次分析法应用于社团活动评估
- 五维指标体系覆盖活动全生命周期
- 权重可配置，适配不同学校需求

**（2）情感可视化创新**
- NLP情感分析结果多维度呈现
- 词云图展示高频关键词
- 情感趋势曲线追踪评价变化

**（3）预测性管理创新**
- 从被动响应转向主动规划
- LSTM预测模型提前识别风险
- 智能推荐优化资源配置

### 5.3 模式创新

**（1）数据驱动决策模式**
- 传统：经验判断 → 本系统：数据分析
- 提供可视化报表辅助决策
- 历史数据沉淀形成知识库

**（2）闭环管理流程**
- 策划 → 执行 → 评价 → 分析 → 改进
- 每个环节数据留痕，可追溯
- 评价数据反哺策划阶段

**（3）多方协同机制**
- 学生、社团、管理部门三端协同
- 信息共享减少沟通成本
- 权限分级保障数据安全
