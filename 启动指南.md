# 校园社团系统 - 完整启动指南

## 一、环境要求

| 组件 | 版本 | 说明 |
|------|------|------|
| Java | 21+ | 后端运行环境 |
| Maven | 4.0+ | 项目构建工具 |
| PostgreSQL | 14+ | 数据库（已安装在 F:\PostgreSQL） |
| Python | 3.13 | AI 算法服务 |
| Node.js | 18+ | 前端运行环境 |
| pnpm | 8.15+ | 前端包管理器 |

**数据库连接信息：**
- 主机: localhost
- 端口: 5432
- 用户名: postgres
- 密码: 123456
- 数据库: campus_club

---

## 二、启动步骤

### 步骤 1：检查数据库是否运行

打开命令行，执行：

```bash
F:\PostgreSQL\bin\psql -h localhost -p 5432 -U postgres -c "SELECT version();"
```

输入密码：`123456`

如果显示 PostgreSQL 版本信息，说明数据库已启动。

**如果数据库未启动**，检查 PostgreSQL 服务或重新启动电脑。

---

### 步骤 2：启动 Java 后端（端口 8080）

**打开第一个命令窗口**，执行：

```bash
cd campus-main
F:\maven\apache-maven-4.0.0-rc-5\bin\mvn spring-boot:run -Dspring-boot.run.profiles=local
```

**启动成功标志：**
```
Tomcat started on port 8080 (http) with context path '/api'
Started CampusClubApplication in X.XXX seconds
```

**验证地址：** http://localhost:8080/api/actuator/health

**API 文档：** http://localhost:8080/api/swagger-ui.html

---

### 步骤 3：启动 Python AI 服务（端口 8000）

**打开第二个命令窗口**，执行：

```bash
cd campus-ai
uvicorn main_simple:app --reload --port 8000
```

**启动成功标志：**
```
Uvicorn running on http://127.0.0.1:8000
Application startup complete.
```

**验证地址：** http://localhost:8000/docs

---

### 步骤 4：启动前端（端口 5173 或自动分配）

**打开第三个命令窗口**，执行：

```bash
# 首次运行需要安装依赖（已安装过可跳过）
cd campus-frontend
pnpm install

# 启动社团端
pnpm --filter club dev
```

**启动成功标志：**
```
VITE v4.x.x  ready in XXX ms

➜  Local:   http://localhost:5173/
➜  press h + enter to show help
```

**访问地址：** http://localhost:5173

## 五、常见问题

### 1. 端口被占用

如果提示端口已被占用，修改启动参数：

```bash
# Java 后端修改端口
mvn spring-boot:run -Dspring-boot.run.jvmArguments="-Dserver.port=8081"

# Python 服务修改端口
uvicorn main_simple:app --reload --port 8001

# 前端修改端口（在 vite.config.ts 中配置）
```

### 2. 数据库连接失败

检查数据库服务是否启动：

```bash
F:\PostgreSQL\bin\pg_isready -h localhost -p 5432
```

### 3. Python 依赖问题

如果 AI 服务启动失败，确保安装了兼容 Python 3.13 的依赖：

```bash
pip install fastapi uvicorn pydantic==2.10.6 loguru httpx python-jose
pip install python-dotenv redis jieba pyyaml
pip install scikit-learn==1.6.1 numpy pandas
pip install Pillow prometheus-fastapi-instrumentator python-multipart
```

### 4. 前端依赖问题

如果前端启动失败，尝试清理缓存：

```bash
cd campus-frontend
rm -rf node_modules
pnpm install
```

## 八、数据库表说明

系统启动后会自动创建以下表（由 JPA Hibernate 自动建表）：

- `users` - 用户表
- `clubs` - 社团表
- `activities` - 活动表
- `activity_participants` - 活动参与记录
- `resources` - 资源表
- `resource_applications` - 资源申请表
- `evaluations` - 活动评价表
- `fund_applications` - 资金申请表

查看所有表：

```bash
F:\PostgreSQL\bin\psql -h localhost -p 5432 -U postgres -d campus_club -c "\dt"
```
